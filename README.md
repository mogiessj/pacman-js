# Pacman en JavaScript Nativo

Este proyecto es una recreación interactiva del juego Pacman, desarrollado utilizando HTML, CSS y JavaScript.

El objetivo de este proyecto es profundizar en conceptos de desarrollo de videojuegos web como loops de renderizado, detección de colisiones, manejo de estados y manipulación del DOM.

## Créditos y Atribución

Este proyecto se ha desarrollado con fines educativos y de aprendizaje personal.

* **Inspiración y base técnica:** Este código está basado y guiado por el tutorial de Youtube de [Code Fiction] - "[ASMR Programming - Coding Pacman - No Talking]" (puedes ver el video aquí [https://www.youtube.com/watch?v=GXlckaGr0Eo]).
